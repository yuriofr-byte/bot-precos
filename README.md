# Bot de Monitoramento de Cambio

Bot que monitora a cotacao de moedas em tempo real e envia alertas por email.

## Funcionalidades

- Monitoramento em tempo real do Dolar/Real
- Alerta por email quando o preco alvo e atingido
- Verificacao automatica a cada hora

## Tecnologias

- Python 3
- Requests
- Schedule
- AwesomeAPI

## Como usar

1. Instala as dependencias: pip3 install requests schedule
2. Configura o email e preco alvo no ficheiro bot_precos.py
3. Executa: python3 bot_precos.py
